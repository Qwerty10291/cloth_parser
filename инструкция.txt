        Установка библиотек
pip install requests lxml cssselect

        Запуск
python main.py

Все изображения сохранятся в директории images

Обратите внимание: из за того что в скрипте используется оптимизация по нескольким процессам,
при принудительном завершении работы спаршенные данные в json файле не сохранятся, нужно дождаться окончания работы скрипта.
